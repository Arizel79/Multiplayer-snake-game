Адресная строка: Видима Высота окна: 0px

Скрытие адресной строки в Chrome на Android

Используйте эти методы для создания более иммерсивного веб-опыта

Метод 1: Мета-тег viewport

Добавьте этот мета-тег в секцию head вашего HTML:

<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">

Метод 2: Fullscreen API

Используйте JavaScript Fullscreen API для перехода в полноэкранный режим:

function requestFullscreen() {
  if (document.documentElement.requestFullscreen) {
    document.documentElement.requestFullscreen();
  } else if (document.documentElement.webkitRequestFullscreen) {
    document.documentElement.webkitRequestFullscreen();
  } else if (document.documentElement.msRequestFullscreen) {
    document.documentElement.msRequestFullscreen();
  }
}

Метод 3: Прокрутка страницы

Адресная строка автоматически скрывается при прокрутке страницы вниз.

window.addEventListener('scroll', function() {
  // Адресная строка скроется автоматически при прокрутке
  updateAddressBarStatus();
});

Метод 4: Установка высоты окна

Используйте CSS, чтобы установить высоту 100vh для элементов:

body, html {
  height: 100%;
  min-height: 100vh;
  /* Мобильные браузеры */
  min-height: -webkit-fill-available;
}

Дополнительная информация

Для достижения лучшего результата комбинируйте эти методы. Помните, что некоторые методы (например, Fullscreen API) требуют действия пользователя для активации.
